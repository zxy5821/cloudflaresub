优选IP订阅生成器

只做三件事：导入自建节点、批量替换为优选 IP、输出可导入主流客户端的订阅链接。

适合场景

3X-UI / Xray / VPS 自建节点 + 你自己找好的 Cloudflare 优选 IP。

重点兼容自建 VMess / VLESS / Trojan 的 WS + TLS 常见写法。

格式为 原名称=新名称，只改等号右侧即可；按原始节点名称匹配。

支持 IP、IP:端口、域名、域名:端口，# 后面可写备注。

可用变量：{name}、{remark}、{index}、{prefix}。

说明

  • 不会帮你找优选 IP，只负责把你现成的优选 IP 替换进节点。
  • 生成的链接是“同一份节点内容”的不同导出格式，分别给不同客户端使用。
  • 推荐勾选“保留原 Host / SNI”，这样更适合 Cloudflare CDN 场景。
  • 订阅链接里的节点信息会通过服务端密钥加密，请不要把链接公开分享。

客户端怎么用

  • V2rayN： 用原始订阅链接
  • Shadowrocket： 用原始订阅链接
  • Clash / Mihomo： 用 Clash 订阅链接
  • Surge： 用 Surge 订阅链接
